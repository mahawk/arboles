¿Qué es un árbol binario?

Un árbol binario es aquel tipo de árbol en el cual sus
nodos tienen grado externo de 0, 1 o 2, o sea hay de 0 a 2 nodos que salen de
este.

Su grado interno es de 0 para la raíz (ningún nodo llega a este) y para los demás
nodos el grado es de 1.

Partes de un árbol binario

Raíz


Es el nodo que tiene grado interno 0, o sea, no
hay nodos arriba que lleguen a él.

Nodo ramal


Es el cual su grado externo es mayor a 0.

Hojas


Son las que tienen grado externo 0, no tienen
nodos 《hijos》.




Recorridos de un árbol binario

Top-down
Para hacer este recorrido se divide el árbol en niveles y se va de la parte superior a la inferior, de izquierda a derecha.
bottom-up
Similar al Top-Down, igual se divide en niveles para mejor facilidad. La diferencia es que este se hace de la parte inferior a la superior de izquierda a derecha.
pre-orden
La regla para este recorrido es que primero se visita el nodo central, luego el sub-nodo izquierdo y por último el sub-nodo derecho.
in-orden
Para ejecutar este recorrido, primeramente, se visita el sub-nodo izquierdo, después el nodo central, y por último el sub-nodo derecho. 
post-orden
Al efectuar este recorrido, primero se debe visitar el sub-nodo derecho, luego el sub-nodo izquierdo y por último el nodo central.
¿Qué es recorrido de un árbol binario?
Proceso de visitar de una manera sistemática, exactamente una vez, cada nodo en una estructura de datos de árbol (examinando y/o actualizando los datos en los nodos)
